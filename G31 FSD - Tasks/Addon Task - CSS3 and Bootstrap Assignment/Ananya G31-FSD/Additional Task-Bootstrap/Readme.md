# CSS3 and Bootstrap 20 Use Cases Assignment

## Project Overview

This project contains 20 simple UI use cases built with HTML5, CSS3, Bootstrap 5, and basic JavaScript.

The pages are written in a beginner-friendly style so students can learn one component at a time.
Each HTML file is a separate example page and does not link to the other pages.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
