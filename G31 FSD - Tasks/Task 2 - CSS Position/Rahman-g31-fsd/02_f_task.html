IMPORTANT RULE =- z-index works on positioned elements (for example, elements with position: relative, absolute, fixed, or sticky). If an element remains position: static (the default), z-index generally has no effect.

Z-INDEX PROPERTY =- The z-index property controls the stacking order of overlapping elements on a webpage. You can think of the page as a stack of layers. When two or more elements overlap, the element with the higher z-index value appears in front of the element with the lower value. For example, if a red box has z-index: 1 and a blue box has z-index: 2, the blue box will appear above the red box when they overlap. Similarly, if a green box has z-index: 3, it will appear above both the blue and red boxes
