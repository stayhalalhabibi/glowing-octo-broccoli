# CSS Position Assignment

## Part A
Static position ignores top, left, right and bottom properties.

## Part B
Relative position moves the element from its original position while keeping its original space reserved.

## Part C
Absolute positioning places an element relative to its nearest positioned parent.

## Part D
Fixed elements remain visible during scrolling.

## Part E
Sticky elements behave like relative until a scroll threshold is reached, then act like fixed.

## Part F
z-index controls stacking order. Higher z-index appears above lower z-index.

## Challenge Questions

1. Default position value: static
2. Removes element from flow: absolute, fixed
3. Visible during scrolling: fixed
4. Requires top/left/right/bottom: sticky
5. Same z-index: later element in HTML appears on top
6. Use absolute when exact placement is needed

Examples:
- Static: Paragraph
- Relative: Banner
- Absolute: Notification badge
- Fixed: Chat button
- Sticky: Navigation bar
