# Challenge Questions

28. What is the default value of the position property?
The default value of the position property is static.

29. Which position type removes an element from the normal document flow?
absolute and fixed positioning remove an element from the normal document flow.

30. Which position type keeps an element visible during scrolling?
fixed positioning keeps an element visible while scrolling.

31. Which position type requires a top, left, right, or bottom value to create a sticky effect?
sticky positioning requires a value like top: 0 to work properly.

32. What happens when multiple elements have the same z-index value?
If multiple elements have the same z-index value, the element written later in the HTML code appears on top.

33. When would you use absolute positioning instead of relative positioning?
I would use absolute positioning when I want to place an element at a specific location inside a parent container, like a notification badge on an image.

34. Give one real-world example of each position type.
Static: Normal paragraph text on a webpage.
Relative: A banner moved slightly from its original position.
Absolute: Notification badge on an image.
Fixed: Chat button or Back to Top button.
Sticky: Navigation bar that stays at the top while scrolling.
