Example 1: Moving a Relative Box

Relative Box

The box is moved:

  • 40px to the right using left: 40px
  • 20px downward using top: 20px

Even though the box moves visually, its original space is still reserved in the layout.

Example 2: Two Adjacent Boxes

Box A
Box B

Box A is moved 50px to the right using relative positioning.

Notice that Box B does not move into Box A's original space because relative positioning keeps the original space reserved.

Observation

  • position: relative moves the element from its original position.
  • The element remains part of the normal document flow.
  • The original space is still occupied.
  • Nearby elements behave as if the moved element never changed position.
