# LearnHub CSS Position Assignment

## Overview

This project is a complete HTML5/CSS3 webpage built for the CSS Position assignment. It demonstrates all five CSS position values within a modern online learning platform theme called **LearnHub**.

## Files

- `index.html` - Complete landing page with semantic structure.
- `style.css` - Full styling with comments and responsive layout.
- `Img1.jpg` through `Img7.jpg` - Image assets used throughout the page.

## Features

- **Static position** example in the hero course preview cards.
- **Relative position** example with a promotional banner and adjacent promotional card.
- **Absolute position** example with a `NEW` badge inside a featured course card.
- **Fixed position** example with a fixed header and support chat button.
- **Sticky position** example with a navigation bar that sticks as you scroll.
- **z-index** stack order demonstration using overlapping colored boxes.
- **Course listings**, **testimonials**, **FAQ**, and **challenge questions** sections.
- **Responsive design** for mobile and tablet screens.

## How to Run

1. Open the `Assignment-2` folder in VS Code.
2. Launch `index.html` using Live Server or open it directly in a browser.
3. Scroll through the page to see fixed and sticky elements in action.

## Notes

- The page is built with semantic HTML tags and modern CSS layout techniques.
- The project includes an explanation section and a summary table describing each CSS position type.
- The `style.css` file contains comments explaining why each position type was chosen.
