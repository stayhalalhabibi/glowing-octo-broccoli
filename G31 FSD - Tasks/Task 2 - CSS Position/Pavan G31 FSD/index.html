FAQ

Answers to the most common questions about this assignment layout.

Why use a fixed header?

Fixed headers keep navigation visible even when users scroll down the page.

What makes sticky navigation different?

Sticky navigation stays in place after the user scrolls past it, while fixed navigation stays always visible.

When should absolute positioning be used?

Use absolute positioning for badges and overlays inside a positioned container.

Challenge Questions

1. The default value of the position property is static.

2. absolute removes an element from the normal document flow.

3. fixed keeps an element visible during scrolling.

4. sticky requires a top, left, right, or bottom value to stick.

5. When multiple elements share the same z-index, the later element in HTML order appears on top within that stacking context.

6. Use absolute positioning instead of relative when you want an element to be placed precisely inside a container without affecting other elements’ layout.

7. Real-world examples:

  • Static — article paragraphs in a blog post.
  • Relative — a badge moved slightly on a card.
  • Absolute — a notification badge on a product card.
  • Fixed — a chat button or sticky header.
  • Sticky — a navigation bar that sticks after scrolling.
