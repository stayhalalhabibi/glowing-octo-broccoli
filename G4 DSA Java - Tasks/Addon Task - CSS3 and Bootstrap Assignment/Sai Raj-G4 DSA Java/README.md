\# SmartBank - Online Banking Dashboard



\## Project Overview



SmartBank is a responsive online banking dashboard built using HTML5, CSS3, Bootstrap 5, and JavaScript. It demonstrates common banking UI components and fulfills the CSS3 \& Bootstrap assignment requirements.



\## Technologies Used



\- HTML5

\- CSS3

\- Bootstrap 5

\- JavaScript



\## Features



\- Responsive Navigation Bar

\- Hero Section

\- Dashboard Summary Cards

\- Recent Transactions Table

\- Money Transfer Form

\- Customer Profile

\- Banking Services

\- Credit Cards

\- Financial Goals

\- Notifications

\- FAQ Accordion

\- Customer Reviews

\- Contact Support Form

\- Responsive Design



\## Folder Structure



```

SmartBank/
```

\## How to Run



1\. Download the project.

2\. Open `index.html` in any modern web browser.

3\. Ensure all images are placed inside the `images` folder.
