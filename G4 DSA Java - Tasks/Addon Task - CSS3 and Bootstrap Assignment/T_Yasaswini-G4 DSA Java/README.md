# College Management Portal

## Project Overview

The College Management Portal is a responsive web application developed using HTML5, CSS3, Bootstrap 5, and JavaScript. It demonstrates various Bootstrap components and CSS3 features commonly used in modern web development.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features Implemented

1. Responsive Navigation Bar
2. Hero Section
3. Student Login Form
4. Student Registration Form
5. Faculty Profile Card
6. Course Card
7. Department Section
8. Contact Form
9. Responsive Student Table
10. Dashboard Statistics Cards
11. Alert Messages
12. Modal Popup
13. FAQ Accordion
14. Progress Bars
15. Tabs Section
16. Breadcrumb Navigation
17. Pagination
18. Fee Summary Card
19. Admin Dashboard Layout Elements
20. Responsive Design
