3

Explanation:

The badge (number 3) is positioned absolutely on top of the image. It uses the .image-container as its positioning context.

When a parent container has position: relative, the absolutely positioned child is placed relative to that parent.

If the parent container has NO positioning context (position property), the absolutely positioned element will reference the nearest ancestor with a position property, or the document body.
