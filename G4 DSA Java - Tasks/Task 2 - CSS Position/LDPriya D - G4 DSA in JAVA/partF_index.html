Task 2 - Part F: z-index

Understanding z-index

This example demonstrates how the z-index property controls the stacking order of overlapping elements.

Overlapping Boxes

Red Box

z-index : 1

Blue Box

z-index : 2

Green Box

z-index : 3

Observation and Explanation

  • The blue box appears above the red box.
  • The green box appears above both the blue and red boxes.
  • A higher z-index value places an element above elements with lower values.
