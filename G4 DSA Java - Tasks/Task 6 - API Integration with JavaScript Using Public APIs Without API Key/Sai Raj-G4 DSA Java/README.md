\# Pokémon Search App



\## Project Overview



This project is built using HTML, CSS, and JavaScript.



It fetches Pokémon data from the PokéAPI using JavaScript fetch().



\## API Used



PokéAPI



https://pokeapi.co/api/v2/pokemon/{name or id}



\## Features



\- Search Pokémon by Name

\- Search Pokémon by ID

\- Display Pokémon Image

\- Display Name

\- Display ID

\- Display Height

\- Display Weight

\- Display Type

\- Display Abilities

\- Display Stats

\- Loading Message

\- Error Handling

\- Responsive Design



\## API Response Fields Used



\- id

\- name

\- height

\- weight

\- abilities

\- types

\- stats

\- sprites.front\_default



\## Technologies Used



\- HTML

\- CSS

\- JavaScript

\- Fetch API
