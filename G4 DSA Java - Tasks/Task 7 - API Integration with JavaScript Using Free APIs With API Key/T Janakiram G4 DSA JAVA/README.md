# NASA Astronomy Picture of the Day App

An application built with HTML, CSS, and vanilla JavaScript that interfaces directly with NASA's Planetary APOD Engine.

## Setup Instructions

1. Clone or download this repository onto your machine.
2. Locate the project root path and create a completely blank file named `config.js`.
3. Paste the following snippet containing your private credential configuration key into `config.js`:

```js
const API_KEY = "YOUR_ACTUAL_SECRET_NASA_API_KEY";
